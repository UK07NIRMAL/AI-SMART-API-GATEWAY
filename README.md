# 🚀 NeuroGate AI — Smart API Gateway

An intelligent backend system built with FastAPI that handles API requests with advanced security, rate limiting, and machine learning-based fraud detection.

---

## ⚡ Features

- 🔐 JWT Authentication
- 🔑 API Key System
- 🧠 ML-based Fraud Detection (Isolation Forest)
- 🚦 Dynamic Rate Limiting (Redis)
- 📊 Activity Tracking (PostgreSQL)
- 👥 Role-Based Access Control (RBAC)

---

## 🏗 Tech Stack

- FastAPI
- PostgreSQL
- Redis
- Scikit-learn
- SQLAlchemy

---

## 📌 Architecture

Request → Middleware → Rate Limiter → Auth → ML Detection → Response

---

## 🚀 Setup

```bash
git clone <your-repo>
cd project

pip install -r requirements.txt

uvicorn app.main:app --reload
```
